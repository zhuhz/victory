# victory
This a simple php mvc framework.

假设项目名为`victory`，MVC的框架命名为`Easyphp`，目录结构为：<br>
victory 			WEB部署目录<br>
|--application		应用目录<br>
|  |--controllers	控制器目录<br>
|  |--models		模块目录<br>
|  |--views			视图目录<br>
|--config			配置文件目录<br>
|--system			框架核心目录<br>
|--static			静态文件目录<br>
|--index.php		入口文件<br>
